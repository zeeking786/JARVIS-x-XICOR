### ⚡ Special Note for PyAudio Installation:
- PyAudio must be installed manually because it is not directly available via `pip install`.
- Follow these steps:
  1. Visit [this link](https://www.lfd.uci.edu/~gohlke/pythonlibs/).
  2. Search for **PyAudio**.
  3. Download the appropriate `.whl` file according to your system architecture:
     - For 64-bit:  
       `pip install PyAudio‑0.2.11‑cp37‑cp37m‑win_amd64.whl`
     - For 32-bit:  
       `pip install PyAudio‑0.2.11‑cp34‑cp34m‑win32.whl`
  4. Then install it using pip.
